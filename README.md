# 🤖 AI Job Hunter

Projeto que utiliza Node.js para buscar vagas de emprego automaticamente em múltiplas APIs e classificá-las com base na compatibilidade com meu currículo.

## 🚀 Funcionalidades

* Buscar vagas automaticamente
* Integrar múltiplas APIs de empregos
* Filtrar vagas Junior
* Calcular score de compatibilidade com o currículo
* Interface simples em HTML

## 🧠 APIs utilizadas

* Remotive
* RemoteOK
* Adzuna

## 🛠 Tecnologias

* Node.js
* Express
* JavaScript
* HTML
* APIs REST

## ▶️ Como rodar o projeto

1. Instalar dependências

npm install

2. Iniciar o servidor

node server.js

3. Abrir no navegador

Abrir o arquivo `index.html`

## 📌 Objetivo

Este projeto foi criado como estudo de automação e integração de APIs para facilitar a busca de vagas de desenvolvimento.
